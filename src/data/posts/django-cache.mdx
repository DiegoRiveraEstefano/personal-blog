# Django Cache

Django Cache is a cache backend for Django that supports multiple backends.

## Installation

```bash
pip install django-cache
```

## Configuration

Add `django_cache` to your `INSTALLED_APPS` setting:

```python
INSTALLED_APPS = [
]
```

Add `django_cache.backends.locmem.LocMemCache` to your `CACHES` setting:

```python
CACHES = {
    "default": {
        "BACKEND": "django_cache.backends.locmem.LocMemCache",
        "LOCATION": "unique-snowflake",
    }
}
```
